Features

🌐 System-wide

Works in any app — browser, editor, chat, terminal.

🔒 100% Local

Ollama runs on your machine. No data ever leaves it.

💾 On-demand

Start Ollama only when needed. Saves ~4GB RAM.

🎯 Tray Icon

System tray indicator to start/stop Ollama with one click.

🖥️ Terminal UI

Full management via dialog/whiptail menu in the terminal.

🌍 i18n

German and English support, auto-detected via $LANG.

Quick Start

1. Install

curl -fsSL https://raw.githubusercontent.com/TheAnonymous/Autocorrect/main/install.sh | bash

2. What gets set up

  • System dependencies (wl-clipboard, wtype, jq, curl, libnotify)
  • Ollama + Gemma 4 model download
  • Keyboard shortcut: Super+Shift+G
  • System tray autostart for Ollama management

3. Use it

Select any text in any application, press Super+Shift+G, and watch the correction appear.
